Módulo 3. Frameworks

En este módulo deberás usar React para el desarrollo de una aplicación web.

Para el desarrollo del módulo deberás revisar los siguientes recursos que se encuentran en la sección de Actividades:

Un video clave: ¿Con o sin framework? Dos webinarios tematicos pregrabados: Transferencia de estado representacional - REST y Creación de interfaces Participar en los tres webinarios tematicos en vivo sobre el desarrollo de aplicaciones web en React JS

Duración: 4 semanas de trabajo Del 11 de octubre al 7 de noviembre de 2021

Objetivo: Reconocer las ventajas de utilizar un framework de JavaScript e integrarlas como herramientas principales para su labor como desarrollador/a.

Desarrollar una aplicación web haciendo uso del framework React.

Inicia con el desarrollo de las actividades del módulo, consulta el material sugerido y completa las pruebas.

El trabajo en equipo iniciará el martes 19 de octubre 2021 en los horarios que notifique tu Scrum máster (profesor o mediador).
